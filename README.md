# Sumit Kumar — Developer Portfolio

My personal portfolio website built from scratch with HTML, CSS and JavaScript. No frameworks, no build tools — just clean vanilla code. Designed to showcase my skills and projects to recruiters and interviewers.

🚀 **Live Demo**

🔗 **https://sumitkumar98313.github.io/portfolio** ⚠️ Hosted on GitHub Pages — loads instantly, no wake-up time

---

## ✨ Features

### 🎨 Design & Layout
- Dark themed UI with a fixed sidebar navigation
- Sidebar collapses into a top navigation bar on mobile
- Smooth scroll behavior between sections
- Clean typography using Outfit and JetBrains Mono fonts

### ⚡ Animations
- Typing animation that cycles through my developer roles
- Skill bars that animate when scrolled into view
- Cards and sections that fade in with staggered entrance effects
- Floating tech chip badges animated around the profile photo
- Rotating dashed ring around the profile avatar

### 📱 Responsive Design
- Works on all screen sizes — mobile, tablet and desktop
- Sidebar becomes a horizontal top nav on smaller screens
- Floating chips hidden on mobile for a cleaner look
- Buttons stack vertically on small screens

### 🧭 Navigation
- Active nav link highlights automatically based on current section
- IntersectionObserver used for scroll-based section detection
- No JavaScript frameworks — everything built with vanilla JS

## 🗄️ JavaScript Modules

**Typing Effect**
- Cycles through developer role titles
- Realistic typing and deleting with variable speed
- Loops infinitely with pause between roles

**Active Nav Observer**
- Uses IntersectionObserver API
- Detects which section is currently in view
- Updates sidebar nav active state automatically

**Skill Bar Animation**
- Observes when skill section enters viewport
- Animates each bar from 0% to its target width
- Runs once per page load, not on every scroll

**Entrance Animations**
- Adds reveal class to cards and section headers
- Staggered delay based on element index
- Smooth fade up transition on scroll into view
